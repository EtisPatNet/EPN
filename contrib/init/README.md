Sample configuration files for:
```
SystemD: etispatnetd.service
Upstart: etispatnetd.conf
OpenRC:  etispatnetd.openrc
         etispatnetd.openrcconf
CentOS:  etispatnetd.init
macOS:   org.etispatnet.etispatnetd.plist
```
